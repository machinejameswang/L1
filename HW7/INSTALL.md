# HW7 Installation Guide

## Copy to your Windows folder

1. Download and unzip `HW7_Boston_Housing_Project.zip`
2. Copy all files into:

```text
D:\AI人工智慧\Antigravity\HW7
```

## Run

```bat
cd /d D:\AI人工智慧\Antigravity\HW7
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

## Streamlit

```bat
streamlit run streamlit_app.py
```
