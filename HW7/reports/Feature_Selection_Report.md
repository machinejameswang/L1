# Feature Selection Report

## Methods Used
This project follows the HW6 style and applies five feature selection methods:

1. Pearson Correlation
2. Mutual Information
3. Recursive Feature Elimination
4. Lasso Regression
5. Random Forest Feature Importance

## Voting Strategy
Each method selects the top features. A feature receives one vote when it appears in the top-k result of each method.
